# 游戏大厅 Bot

基于 Socket.io 的游戏大厅服务，为用户提供 4 款小游戏。

## 快速开始

1. **配置环境变量**
   ```bash
   cp .env.example .env
   ```
   编辑 `.env` 文件，设置 `BOT_PASSWORD`

2. **启动 Bot**
   ```bash
   node bots/game-bot/index.js
   ```

## 功能特性

- ✅ Bot 在线时，所有用户可见
- ✅ Bot 下线时，自动隐藏所有游戏房间
- ✅ 支持 4 款游戏：数字华容道、记忆翻牌、速算挑战、古诗背诵
- ✅ 游戏进度服务器端保存

## 使用方法

1. 用户点击"游戏大厅"创建私聊
2. 进入房间后自动显示游戏选择卡片
3. 点击游戏卡片开始游戏
